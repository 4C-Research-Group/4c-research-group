# 🎯 4C Research Lab Website - Handover Guide

## 👋 Welcome to Your New Website!

This guide is specifically designed for the 4C Research team members who will be managing the website. **No coding knowledge required!**

## 🚀 Quick Start (5 Minutes)

### 1. Access Your Website

- **Website URL**: Your website will be live at your domain
- **Admin Login**: Use the admin dashboard to manage content

### 2. Admin Dashboard Access

- **Email**: admin@example.com
- **Password**: (provided separately)
- **Login URL**: `your-website.com/admin`

## 📝 What You Can Do (No Coding Required!)

### 🏠 **Update Page Content**

1. Go to Admin Dashboard → "Edit Pages"
2. Click on any page (Home, About, etc.)
3. Edit text, images, and content
4. Click "Save" - changes appear immediately!

### 📰 **Manage Blog Posts**

1. Go to Admin Dashboard → "Blog"
2. **Create New Post**: Click "Create Blog"
3. **Edit Existing**: Click "Edit" on any post
4. Add images, text, tags, and categories
5. Publish instantly!

### 👥 **Manage Team Members**

1. Go to Admin Dashboard → "Team"
2. **Add New Member**: Click "Add Team Member"
3. **Edit Existing**: Click "Edit" on any member
4. Update photos, roles, and information
5. Add testimonials

### 🔬 **Update Research Projects**

1. Go to Admin Dashboard → "Projects"
2. **Add New Project**: Click "Add Project"
3. **Edit Existing**: Click "Edit" on any project
4. Update descriptions, images, and status
5. Keep research current!

#### 📋 **Project Management Guide**

**Project Actions:**

- **View**: See how the project appears on the public research page
- **Edit**: Modify basic project details, content with rich text editing, and upload images (main + additional images)
- **Advanced Edit**: Modify detailed project information including objectives, team members, tags, and long descriptions with rich text editing
- **Delete**: Remove projects and all associated images (this action cannot be undone)

**Project Settings:**

- **Status**: Use "upcoming" for future projects, "active" for ongoing, "completed" for finished
- **Categories**: Organize projects by research type (Clinical Research, Implementation Science, Clinical Trial, Basic Science, Quality Improvement)

**Image Management:**

- **Main Image**: Hero image displayed prominently on the project page
- **Additional Images**: Up to 2 additional images displayed in the project gallery
  - Use the plus button to upload multiple images at once
  - Images are automatically resized and optimized
  - Remove individual images using the × button on each preview

## 🎨 Content Management Tips

### 📸 **Adding Images**

- **Size**: Use images around 800x400 pixels for best results
- **Format**: JPG or PNG files work best
- **Upload**: Simply drag and drop or click to upload
- **Free Images**: Use Unsplash.com for professional photos

### ✍️ **Writing Content**

- **Keep it Simple**: Write as you would for colleagues
- **Use Headings**: Break up text with titles and subtitles
- **Add Links**: Include relevant research links
- **Proofread**: Check spelling and grammar

### 🏷️ **Using Tags and Categories**

- **Blog Tags**: Add relevant keywords (e.g., "neuroscience", "research", "brain")
- **Categories**: Group similar content together
- **Search**: Help visitors find your content easily
